# CSS 类名命名汇总

* collapsed 收拢的
* expanded 展开的
* current 当前
* selected 选中的
* active 活跃的
* enabled 可用
* disabled 不可用

* 尾部 footer
* 主栏 main
* 侧栏 side
* 盒容器 wrap
* 条目 item。一般
