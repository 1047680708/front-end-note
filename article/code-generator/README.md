# 如何开发代码生成工具
无需写代码，用[易企秀](eqxiu.com)能做营销页面，用[Strikingly](https://www.strikingly.com/) 能做网站，用。

易企秀，Strikingly 都是代码生成工具。代码生成工具能:
* 降低对开发者的要求。代码生成工具的使用者只需懂业务，不需懂代码。
* 减少重复劳动。
* 提升工作效率。

## 哪些场景适合用代码生成工具
配置项少，出现次数多。
配置项少：1，代码生成代码量少。2，易用。 
出现次数多： 1， 代码生成工具的能节约的时间多。
