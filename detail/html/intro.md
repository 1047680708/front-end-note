## <a name="what">HTML是神马</a>
HTML是标记语言。标记语言是一系列的标签。

## <a name="what">基本结构</a>
网页示例
```
<!DOCTYPE html>
<html>
<head>
    <meta charset="utf-8">
    <title>标题</title>
</head>
<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>
</html>
```
