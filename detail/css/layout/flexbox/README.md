# flexbox
## 历史
如果你正在查找有关于Flexbox的博客资料，你看到了“display:box;”或者“box-{*}”属性，那么你看的正是2009年版本的Flexbox。

如果你正在查找有关于Flexbox的博客资料，你看到了“display:flexbox;”或者“flex()”函数，那么你看的正是2011年版本的Flexbox。
