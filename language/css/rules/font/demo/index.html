字体

字体大小

用 font-size 来设置字体大小。

说明

  • 字体大小会继承。
  • 在所有现代浏览器中，默认的字体大小就是 16px。
  • 基于用户阅读的考虑，浏览器支持的最小的字体值是12px。即使设置小于12px的字，显示的值也是12px。
  • 设置 font-size: 0，字就隐藏了。

示例

长度单位

12px， 14px， 30px。

em和百分比

1em的值是父元素的字体值。100%的值是当前元素的字体值。
当前元素是 12px。 2em也就是24px。这个元素的子元素设置2em 就是 48px。
200%也是24px。
当前元素是 14px。2em也就是28px。

rem

1rem的值是html的字体大小。
1rem, 2rem。

绝对大小值

small, medium, large 。
