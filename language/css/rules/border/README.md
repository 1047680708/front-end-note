# 边框
* border-width
* border-style
* border-color
* border
* [border-image](https://developer.mozilla.org/en-US/docs/Web/CSS/border-image)
* border-radius

如
```
border: 2px solid #f00;
```

box-shadow 和 border-radius 配合，可以做内圆角
```
background: tan;
border-radius: .8em;
padding: 1em;
box-shadow: 0 0 0 .6em #655;
outline: .6em solid #655;
```
