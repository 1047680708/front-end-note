# 新闻列表
做如下图样式。  
![今日头条列表（http://www.toutiao.com/）](toutiao-list.png)

建议分别用 Flex 和 inline-block 来实现。

## 工具
* 量尺寸，量色等可用 Photoshop(功能强大，收费很贵) 或 [Picpick](http://www.picpick.org/en/)(轻量级，功能简单)。
* 图片可以用 [placeholder](https://placeholder.com/)。
