# 浏览器快捷键
* 常用操作
  * Ctrl + F:在网页中查找内容
    * F3:下一个
    * Shift + F3:上一个
  * Ctrl + L：使地址栏获得焦点
  * Ctrl + K：使地址栏旁的搜索栏获得焦点
  * Ctrl + Enter： 自动补全当前地址，并进入
  * Ctrl + D: 将当前页面保存到书签
  * Ctrl + H: 历史
  * Ctrl + J : 打开下载内容
  * Ctrl + Shift+ P:隐私浏览模式（在隱私瀏覽視窗中，Firefox 不會保留任何瀏覽記錄、搜尋記錄、下載記錄、網頁表單輸入內容、Cookie、暫存檔案）
* 缩放页面
  * Ctrl + +:放大
  * Ctrl + -:缩小
  * Ctrl + 0:默认大小
* 标签页操作
  * Ctrl + Shift+ Tab： 激活左边一个标签页
  * Ctrl + Tab ： 激活右边一个标签页
  * Ctrl + [0-8] : 到某个标签
  * Ctrl + 9 ： 选择最后标签页
  * Ctrl + T: 新建Tab
  * Ctrl+ shift + T ：打开最近关闭的一个tab
  * Ctrl + w : 关闭当前tab
  * Ctrl + Shift + A：打开附加组件
* 网址搜索（在地址栏中输入）
  * * + 空格 + 关键字：搜索满足条件的收藏的网站
  * ^ + 空格 + 关键字：搜索满足条件的浏览历史
  * % + 空格 + 关键字：搜索满足条件的打开的标签
